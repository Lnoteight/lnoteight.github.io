状态表示

声调

日本語の森

  • 平板型（平板式，0调）：第一个音节低，后续音节全部高。
    • 例：さくら（桜） [0调] → 低-高-高
    • 例：ともだち（友達） [0调] → 低-高-高-高
  • 头高型（头高式，1调）：第一个音节高，后续音节全部低。
    • 例：はな（花） [1调] → 高-低
    • 例：にほん（日本） [1调] → 高-低-低
  • 中高型（中高式，2~N调）：第一个音节低，第二个音节开始升高，在中间某个音节后下降。
    • 例：やま（山） [2调] → 低-高-低
    • 例：おとこ（男） [3调] → 低-高-高-低
  • 尾高型（尾高式，特殊）：第一个音节低，后续音节逐渐升高，最后一个音节下降（多用于后续接助词的情况）。
    • 例：いもうと（妹） [尾高型] → 低-高-高-高-低（+助词：いもうとが）

な形容词

「嫌い」是极少几个以 「い」而非汉字结尾的な形容词之一，这是因为「嫌い」实际上是从动词「嫌う」派生而来的。

名词|な形容词 +「だった」表过去状态

い形容词

所有い形容词都是以平假名「い」结尾的

い形容词以「い」结尾的全部意义就在于活用的时候改变「い」而汉字不动

副词

如何将形容词变为副词

  • 对い形容词： 把「い」换成「く」。

    例： 早い → 早く

    まやかは朝ご飯を早く食べた。
    摩耶花很快的吃了早饭。

  • **对な形容词：**后面加「に」。

    例： きれい → きれいに

    まやかは自分の部屋をきれいにした。
    摩耶花打扫了她的房间。

  • 不是所有副词都是从形容词变化而来的，比如「全然」和「たくさん」本身就是副词

    映画をたくさん見た。
    看了很多电影。

    最近、全然食べない。
    最近什么都没吃。

ない形

例外

  1. する → しない
  2. くる → こない
  3. ある → ない

tip

  • ある （う动词） - 存在（无生命）
  • いる （る动词） - 存在（有生命）
  • **る动词：*把「る」去掉换成「ない」
    例：食べる + ない = 食べ    ない*
  • 以「う」结尾的う动词： 把「う」换成「わ」，再加上「ない」
    例：買う + わ + ない = 買わない
  • **其他う动词：*把末假名换成同行的あ段假名，再加上「ない」
    例：待つ + た = 待    たない*

他动词和自动词

  • 他动词: 由一个活动主体完成的动作

    電気を消す。

  • 自动词: 没有直接主体的情况下发生的动作

    自动词不能带直接对象，因为没有直接的动作主体

    例: 電気を消える。
    （「を」应该换成「が」或者「は」）

助词

はがも

  • は　主题提示, 强调句子的整体，常用于已知信息或背景信息。

  • が 主语提示，强调主语本身，常用于引入新信息或强调特定对象

を

  • 单词后面加「を」表示它是动作的直接作用对象

  • 「を」本来念作 /wo/，但真实生活中一般念 /o/

  • 地点也可以作为「歩く」和「走る」这种动作动词的直接对象。因为动作动词是对地点做的

    1. 街をぶらぶら歩く。
      无目标的在街上闲逛。（字面：无目标的走街）
    2. 高速道路を走る。
      跑在高速路上。（字面：跑高速路。）

  • 合用名词和「する」时，助词「を」可以省略，你可以把整个 [ 名词 + する] 当作一个动词

    1. 毎日、日本語を勉強する。
      每天学习日语。
    2. メールアドレスを登録した。
      注册了电邮地址。

に (目标助词)

与 を 的区别

  • 「に」 朝向对象

  • 「を」 直接作用于对象

    1. 猫は部屋にいる。
      猫在房间里。
    2. いすが台所にあった。
      椅子曾经在厨房里。
    3. いい友達に会った。
      见过了好朋友。
    4. ジムは医者になる。
      Jim 将成为医生。
    5. 先週に図書館に行った。
      上周去过了图书馆。

へ(方向助词)

  • 「へ」读音是 /he/，但作为助词时读成 /e/ （え）

  • 「へ」表示朝着某个方向 而「に」表示朝着最终的目标(存疑)

  • 可以表示抽象概念, 用于谈论向着特定的未来目标或期望努力

    1. ボブは日本へ行った。
      Bob 往日本去了。
    2. 家へ帰らない。
      不是朝着家的方向去。
    3. 部屋へくる。
      向着房间走来。
    4. 勝ちへ向かう。
      向着胜利进发。

  • 如果没有具体方向的话是不能用「へ」的

    医者へなる。
    （「医者になる」的语法错误版本。）

で(上下文助词)

  • 助词「で」指明了动作发生的上下文
  • 「で」的意思可以理解成「通过、借助于」

与「何」配合

  • 「何」大部分时候读 「なに」, 另读「なん」(暂时忽略)

    1. 何できた？
      通过什么来的？
    2. バスできた。
      通过巴士来的。

  • 「何で」(为什么) 是一个独立单词, 读作「なんで」 , 与「で」无关

    1. 何できた？
      你为什么来？
    2. 暇だから。
      因为我有空。

と(包含助词)

  • 连接两个名词「和」

  • 表示和某人或某物一起完成了动作

    1. 友達と話した。
      和朋友聊天了。
    2. 先生と会った。
      和老师见面了。

や 、とか(罗列助词)

  • 「や」跟助词「と」一样用来罗列一个或多个名词
  • 比助词「と」模糊很多。它暗示也许有其他没有列出的名词，或者清单里的物件并不都适用于上下文。
  • 「とか」和「や」的意思一样，但更口语化一些。

の

  • 在句尾用表解释, 可理解为「问题是…」

    今は忙しいの（だ）。
    问题是（我）现在有点忙。

    不加だ会有点娘

  • 如果要表达状态，且又要用「の」来体现解释的口吻，我们就要添加「な」来和仅表示「的」的「の」区分开。

    • ジムのだ。
      这是 Jim 的。
    • ジムなのだ。
      这是 Jim 的（解释的口吻）

  • 用来提问, 提问的时候只用「の」是没问题的, 是中性的

    今は忙しいの？
    （你）是不是现在很忙？（中性）

ねよ(结尾助词)

「ね」

  • 希望别人赞同自己的时候，会在句尾加一个「ね」

  • 意思类似「对吧？」、「不是吗？」

  • 「そうね」意思可译成没错, 男性一般会说 「そうだね」

    まやか：いい天気だね。
    摩耶花：好天气，是吧?

    アリス：そうね。
    Alice：是那样， 不是吗?

    まやか：おもしろい映画だったね。
    摩耶花：电影挺好看的，对吧?

    アリス：え？全然おもしろくなかった。
    Alice：啊？不好看，一点也不好看。

「よ」

  • 「よ」在句尾指想要让人知道新的信息, 类似 you know

    まやか：時間がないよ。
    摩耶花：知道不，没时间了。

    アリス：大丈夫だよ。
    Alice：没事，you know.

「よね」

  • 「よね」表示既提示新的信息 也希望赞同

    アリス：ボブは、魚が好きなんだよね。
    Alice：知道不，你喜欢吃鱼，不是吗?

    ボブ：そうだね。
    Bob：这样哈?

注意

  • 主题是地点时(にへで)

  • 可以把主题助词（「は」和「も」）加在三个可以指代地点的助词（「に」、「へ」、「で」）后面

ボブ：学校に行った？
Bob：（你）去学校了吗？

アリス：行かなかった。
Alice：没去。

ボブ：図書館には？
Bob：那图书馆呢？

アリス：図書館にも行かなかった。
Alice：也没去图书馆。

  • 主题是直接对象时(を)

  • 不能使用「をは」　直接使用「は」表示主题

    1. 日本語を習う。
      学日语。
    2. 日本語は、習う。
      关于日语，（会去）学习。

    请不要这么写：

    • 日本語をは、習う。
      （这样不对。）

代词

  • 我
  1. 私 【わたくし】 - 正式场合使用，男女通用。
  2. 私 【わたし】 - 普通场合的礼貌用语，男女通用。
  3. 僕 【ぼく】 - 主要是男性使用，相当讲礼貌或相当随意的场合都可以用。
  4. 俺 【おれ】 - 非常随意的口语，基本只有男性使用。
  5. あたし - 很女性化很口语的说法。但很多女生会用「わたし」，因为「あたし」听起来太娘了。
  6. 自己的名字 - 这也是很女性化同时很幼稚的自我称呼。
  7. わし - 通常是中年大叔使用。
  • 你
  1. あなた - 只有在没办法直接跟人说话，或者不知道别人名字的时候才用。比如纸面上向读者提出而读者又必须回答的问题，就可能会用「あなた」。
  2. 君【きみ】 - 男生对女生亲密而又傲慢的称呼，可能会显得粗鲁。
  3. お前【お・まえ】 - 很粗鲁的称呼对方的方法。通常是男人使用，且会说成「おめえ」。
  4. あんた - 熟人之间很不客气的称呼，说话的人一般是生气了。
  5. 手前【て・めえ】 - 非常粗鲁的说法，类似「お前」，并且更凶，一般会说成「てめ～～」，听起来像是准备开打。我只在动漫和电影里见过这种用法。实际上如果你这么称呼你的朋友，他们反而可能会哈哈大笑，说你看漫画书看多了。
  6. 貴様【き・さま】 - 非常、非常粗鲁的说法。听起来像是你要干掉对方。我还是只在漫画书里见过，这里介绍只是为了让你能看懂漫画书。

语序

  • 一个完整的句子需要在末尾有动词（这里包括隐式的状态表达）

    1. 私は公園でお弁当を食べた。
    2. 公園で私はお弁当を食べた。
    3. お弁当を私は公園で食べた。
    4. 弁当を食べた。
    5. 食べた。

  • 完整的句子（关系从句）可以用来修饰名词，形成嵌套关系从句的句子。但「だ」除外

    お弁当を食べた学生が公園に行った。
    吃过午饭的学生去了公园。
